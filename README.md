# MasterMind

Programme d'un **MasterMind** graphique sous pygame en **Python**.


# Comment jouer ?

Pour jouer, il vous faut avoir **Python** et **Pygame** sur votre machine. Pour installer Pygame, faites la commande suivante depuis le terminal :
> pip install pygame
